1. Создайте массив из 10 строк с именами пользователей (Иванов, Петров, Сидоров, Алексеева, Казанцев, Антропов, Анисимова, Кузнецов, Соловьев, Кошкина). Выведите список пользователей в консоль в алфавитном порядке.
2. Создайте массив из 10 элементов со случайными числами от 0 до 99. Найдите максимальное и минимальное значение этого массива.
3. Создайте класс пользователя User, объекты которого сохраняют имя пользователя и его оценку. Запросите в консоли при помощи метода gets имена и оценки для трех пользователей. С использованием этих данных, создате массив, содержащий три объекта класса User.
4. Создайте класс пользователя User, объекты которого сохраняют имя пользователя и его оценку. Создайте массив из объектов класса User, каждый из которых будет хранить разных пользователей и случайные оценки от 1 до 5. Извлеките из массива пользователя с максимальной оценкой.
4. *Создайте класс Point для точки в двумерной системе координат. Каждый объект такого класса должен сохранять числа x и y. Создайте два объекта класса Point и вычислите расстояние между этими двумя точками по формуле квадрата расстояния.
